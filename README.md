# 基于 ROS2 的工业机器人运动控制系统架构设计与仿真验证

## 项目简介
本项目为本科毕业设计工程实现仓库，研究方向为基于 ROS2 的工业机器人运动控制系统架构设计与仿真验证。
研究对象为 UR5 工业机器人，开发环境基于 Ubuntu 22.04 + ROS2 Humble + Gazebo。

## 当前阶段
- 阶段 0：完成
- 阶段 1：完成
- 阶段 2：进行中（系统总体架构实现）

## 工作空间结构
- src/：ROS2 功能包源码
- docs/：架构、接口、进度与实验文档
- bags/：实验数据
- scripts/：辅助脚本
